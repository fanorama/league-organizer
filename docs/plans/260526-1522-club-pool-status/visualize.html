Phase Timeline

Phase 1 [S] — Schema + Wheel

Tambah status: 'pool' ke semua save() call di teams.js. Update wheel.js: kandidat = pool, assign → set status: 'active'.

Phase 2 [M] — UI Two-Section

Restructure render teams.html: dua section terpisah Peserta Liga dan Pool Referensi dengan count masing-masing. Spin button disabled saat pool kosong.

Phase 3 [S] — Downstream Filters

league.js dan season.js: filter query ke status === 'active' sehingga jadwal dan validasi peserta tidak menyertakan pool clubs.
