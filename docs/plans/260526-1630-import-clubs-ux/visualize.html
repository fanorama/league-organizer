Risks & Open Questions

Auto-load & API limit
Auto-load bisa memicu fetch tiap buka modal untuk competition baru.
Mitigasi: cache 7-hari sudah ada — hanya fetch jika cache expired.
Sticky footer overlap
Footer bisa menutupi item terakhir di list.
Mitigasi: tambah padding-bottom saat footer aktif.
Open: Konfirmasi add?
Asumsi: langsung add & tutup modal tanpa dialog konfirmasi — sesuai pola existing.
Open: Select all scope?
Asumsi: "Select all visible" hanya select yang belum di pool (exclude disabled).
